# Learn CSS

This is a simple project to learn and practice CSS, featuring a website that introduces web technologies like HTML, CSS, JavaScript, and React.

## Description

The website includes:

- **Header**: Logo, navigation menu, and CTA buttons (Sign In/Sign Up).
- **Hero Section**: Introduction to the website with a "Get Started" button.
- **Features**: Sections introducing HTML, CSS, JavaScript, and React with logos.
- **Products**: List of products with discount badges and buy buttons.
- **Blogs**: Articles with images and "Read More" buttons.
- **Footer**: Contact information and copyright.

## Technologies Used

- HTML5
- CSS3 (external stylesheet)
- JavaScript (for future enhancements)

## Project Structure

```
learn-css/
├── index.html          # Main HTML file
├── style/
│   └── style.css       # CSS file
├── images/             # Folder containing images (logos, hero, etc.)
└── README.md           # This file
```

## How to Run

1. Clone the repository: `git clone https://github.com/lephugiau/learn-css.git`
2. Open `index.html` in a web browser (Chrome, Firefox, etc.).
3. Ensure the `images/` folder contains the necessary image files.
